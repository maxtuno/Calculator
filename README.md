Calculator
==========

A simple calculator created in 5 minutes with ANTLR4, an example of a small interpreted language with centralized object. (for fun)

(Public domain, do what you want! play!)

Input (source.cal):

1 + 2

((3 * 2) + 5)

(666 + 777) - 5

a = 5

b = 7

a + b/2

Output:

3.0

11.0

1438.0

8.5
